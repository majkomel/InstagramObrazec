# Instagram Login & Signup Replica

A simple, clean replica of Instagram's login and sign-up pages. This project focuses on front-end design and aims to visually and functionally resemble the original Instagram interface.

##  Features

* Styled form elements (inputs, buttons, etc.)
* Separate pages for login and sign-up

##  Technologies Used

* **HTML** for structure
* **CSS** for styling
* **JavaScript** needed for form interactions

##  Purpose

This project was created for an assignment where the objective was to replicate the Instagram UI as accurately as possible using only front-end technologies.
